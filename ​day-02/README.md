Counter App – Day 2

📘 Week 1 – JavaScript Fundamentals

This project is part of my JavaScript learning journey. On Day 2, I built the basic structure of a Counter App using HTML, CSS, and JavaScript.

🎯 Project Objective

- Create a simple Counter App.
- Connect HTML, CSS, and JavaScript files.
- Understand DOM manipulation.
- Practice JavaScript functions.

🚀 Features

- Display the current counter value.
- Increment button to increase the count.
- Decrement button to decrease the count.
- Reset button to set the counter back to zero.
- Dynamic counter updates using JavaScript.

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

📂 Project Structure

Counter-App/
├── index.html
├── style.css
├── script.js
└── README.md

📖 What I Learned

- Linking HTML, CSS, and JavaScript files.
- Selecting elements using "getElementById()".
- Updating content with "innerText".
- Creating and calling JavaScript functions.
- Managing a counter using variables.
- Building a simple interactive web application.

🎯 Future Improvements

- Add a minimum value limit.
- Disable the decrement button when the counter reaches zero.
- Improve the UI with animations and better styling.
- Make the layout fully responsive.
